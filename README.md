
## Laravel - Task list

The laravel task list allows users to register and manage there tasks when logged in. Users also have the ability to remove tasks and log out. 

 - v1.0.0
 - PHP 7.2
 - Laravel 6.2

## Installation
 - run `composer install`
 - Install migrations `php artisan migrate `
 - run `npm install`
